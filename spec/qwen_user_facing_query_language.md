# Qwen 用户侧查询表达

## 需求

Qwen 徒步助手可以使用工具返回的结构化字段进行查询和判断，但面向用户回答时，
不得直接暴露内部字段名、布尔值、JSON 查询条件或筛选逻辑。

例如工具结果显示没有路线同时满足 `has_toilet: true` 和
`has_supply_shop: true` 时，应回答“目前没有同时有厕所和补给点的路线”，
而不是复述字段名和布尔值。

## 表达规则

- 将内部字段和枚举值转换为自然、简洁的中文。
- 不向用户展示 JSON、字段名、布尔值或内部筛选表达式。
- 无结果时，直接说明哪些用户需求暂时无法同时满足。
- 不根据字段值补充工具结果没有提供的原因，例如不得把 `false` 解释为
  “未通过官方设施认证审核”。

## 验收

- Qwen Agent 系统提示包含用户侧自然语言表达规则。
- 系统提示明确禁止暴露内部查询条件和无依据推断。
