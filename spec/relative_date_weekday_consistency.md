# 相对日期星期一致性

## 背景

用户或外部上下文可能同时给出日期和星期，但二者不一定一致。例如 `2026-06-18`
实际是星期四，如果误写为星期三，继续推算“后天”会连带得到错误星期。

## 需求

- 当前日期上下文必须同时包含 ISO 日期和由程序计算的星期。
- 当前日期上下文必须包含程序核验的“明天”和“后天”日期与星期，作为模型回复前的兜底约束。
- Agent 遇到“今天”“明天”“后天”“本周六”等相对日期表达时，必须调用相对出发日期查询工具。
- 相对日期、具体日期和星期名称必须以工具返回结果为准。
- 当用户给出的日期和星期不一致时，Agent 应指出冲突并按工具返回的日期与星期更正，不得沿用错误星期。

## 验收

- 参考日期为 `2026-06-18` 时，“后天”返回 `2026-06-20（星期六）`。
- 当前日期上下文在参考日期为 `2026-06-18` 时包含 `后天是 2026-06-20（星期六）`。
- Qwen Agent 系统提示包含日期与星期冲突时以工具返回结果为准的规则。
