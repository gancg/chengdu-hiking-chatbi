# 单日往返路线默认推荐

## 背景

当前路线推荐用于成都周边徒步 demo，推荐结果默认应服务于单日往返出行。样例数据中
可能存在 `duration_days > 1` 的路线，但多日游路线的住宿、跨日天气、补给和行程拆分
尚未完整建模，暂不应作为默认推荐结果。

## 需求

- 推荐服务未显式指定最大行程天数时，默认只返回 `duration_days = 1` 的路线。
- Qwen Agent 面向用户推荐时，默认把候选路线说明为单日往返路线。
- 用户请求多日游路线时，Qwen Agent 应说明多日游路线后续补充，当前作为 TODO。
- 显式 API 参数 `max_duration_days` 仍保留，用于未来扩展和内部验证。

## 验收

- 不传 `max_duration_days` 调用推荐服务时，不得返回 `duration_days > 1` 的路线。
- Qwen Agent 系统提示应包含“默认单日往返”和“多日游 TODO”的边界说明。
