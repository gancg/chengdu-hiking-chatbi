# 移动端 H5 聊天内容横向扩展

## 需求

- 扩展 H5 Chatbot 内部聊天内容包装层的可用宽度。
- 将 Gradio 主内容区域默认的左右 `32px` 内边距压缩为 `8px`。
- 减少聊天内容左右两侧的无效留白，使路线信息等长内容获得更多横向空间。
- 仅影响 H5 Chatbot，不改变输入框、快捷提问和桌面 Web 页面。

## 验收

- H5 Chatbot 内的 `.wrap` 宽度为 `100%`，且不受额外最大宽度限制。
- H5 的 `main.fillable` 左右内边距为 `8px`，纵向内边距保持不变。
- `.wrap` 左右内边距压缩为 `5px`。
- 机器人气泡明确使用完整可用宽度，不受内容宽度收缩。
- 机器人头像与气泡之间的间距从组件默认的 `10px` 压缩为 `5px`，将空间让给文字内容。
- 样式选择器必须以 `.h5-chatbot` 为作用域。
