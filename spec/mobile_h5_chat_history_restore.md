# 移动端 H5 聊天记录恢复

## 需求

- H5 使用 Gradio `BrowserState` 将最近一次会话保存在当前浏览器，不采集浏览器指纹。
- 记录保留 12 小时；返回、刷新或重新打开同源 H5 页面时恢复聊天内容和 Agent 上下文。
- 用户消息提交后立即保存，助手完整回复后再次保存。
- 最多保存最近 100 条消息和 512 KB JSON；超限时从最早的完整用户轮次开始裁剪。
- 损坏、过期或版本不兼容的状态必须清空，不得阻止页面加载。
- HTTP/HTTPS 外链在新窗口打开，并设置 `noopener noreferrer`。
- 桌面 WebUI 行为保持不变。

## 配置

- `CHATBI_H5_STORAGE_SECRET`：可选的 Gradio BrowserState 稳定加密密钥。未配置时使用进程级随机密钥，服务重启后旧缓存失效。
