# 点名路线的推荐锁定

## 需求

用户明确点名库内目的地或路线时，后续查询只能围绕该命中路线进行，不得把风景、区域或难度相似的其他路线混入结果。

例如用户说“去爬巴朗山”，命中 `wenchuan-balangshan-panda-peak` 后，推荐工具只能返回该路线，不能再返回二古溪、大二普三海等雪山路线。

## 实现约束

- 推荐服务接受可选 `route_id`，存在时先把候选集合限定为该路线。
- 自然语言推荐工具只使用 `routes.group_tour_search_terms_json` 对用户点名内容做模糊匹配，唯一命中后自动注入 `route_id`。
- `routes.name` 不参与该工具的点名目的地匹配。
- 未找到该 `route_id` 时抛出明确异常，不得回退为全库相似路线推荐。
