# Qwen 流式超时恢复

## 问题

DashScope SDK 的流式请求默认读取超时为 300 秒。连接异常但没有正常结束时，
Web 页面会长时间保持生成中状态，用户无法继续操作。

## 需求

- Qwen 模型调用必须显式设置流式请求读取超时，不得沿用 SDK 的 300 秒默认值。
- 默认超时为 20 秒，可通过 `CHATBI_QWEN_REQUEST_TIMEOUT_SECONDS` 调整。
- 配置值必须是正整数；无效值应在启动时抛出含义明确的异常。
- 超时由既有模型异常和重试流程处理，最终失败时必须保留诊断日志，不得无限等待。

## 验收标准

- 默认构建 Qwen Agent 时，`generate_cfg.request_timeout` 为 20。
- 环境变量可以覆盖默认超时。
- 流式请求超过空闲时限后能够结束等待，并进入既有异常处理流程。
