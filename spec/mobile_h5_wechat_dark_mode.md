# 微信深色模式下 H5 快捷提问可读性

## 背景

微信手机端会跟随系统深色模式改变 Gradio 组件的默认背景色，但 H5 自定义文字颜色仍为深色，导致“快捷提问”标题和问题内容对比度不足。

## 要求

- H5 页面固定使用浅色配色，不跟随微信 WebView 的深色模式改变组件颜色。
- “快捷提问”面板使用明确的浅色背景和深色文字。
- 每条快捷问题按钮使用明确的浅色背景、边框和深色文字。
- 修复仅作用于 H5 页面，不影响桌面 Web 页面。

## 验收标准

- 深色系统主题下通过微信打开 H5 页面，“快捷提问”标题、展开图标和三条问题均清晰可读。
- 输入区域与快捷提问区域保持浅色主题。
