# Qwen 路线选择前置确认

## 背景

路线推荐和后续筛选高度依赖出行方式与出行时间。不同出行方式会影响可选路线、交通耗时和费用估算；出行时间会影响交通估算、节假日判断和天气预警。

## 规则

- Qwen Agent 在和用户沟通选择路线时，必须先明确出行方式和出行时间。
- 出行方式必须在自驾、公共交通、报团中明确其一；用户表达为拼车时，可按非公共交通的结伴出行意图继续确认是否自驾/拼车。
- 出行时间必须明确到具体出发日期；涉及交通估算时还应尽量确认出发时段或出发时间。
- 在出行方式和出行时间都明确之前，不得继续追问预算、强度、风景、设施等其它筛选条件。
- 用户明确选择报团时，使用普通路线推荐工具，并将交通方式设置为 `group_tour`。

## 验收

- Qwen Agent 系统提示包含“先明确出行方式和出行时间，再确认其它筛选条件”的规则。
- 访谈引导在早期轮次要求优先补齐出行方式和出行时间。
- Qwen Agent 系统提示明确报团场景应使用普通路线推荐工具和 `group_tour` 交通方式。
