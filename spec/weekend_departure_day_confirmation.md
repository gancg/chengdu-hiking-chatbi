# 周末出行日期确认

## 背景

交通、天气和路线推荐估算都需要具体出发日期。“周末”可能指星期六或星期日，
“本周末”“下周末”也会对应两个候选日期。若 Agent 未向用户确认就自行选择其中
一天，可能导致节假日、星期和交通估算错误。

## 需求

- 用户只说“周末出行”“周末早上出发”等未明确具体日期的表达时，Agent 必须先确认具体哪一天。
- 用户说“本周末”或“下周末”时，Agent 必须先调用相对出发日期查询工具获得候选日期。
- 当候选日期包含星期六和星期日，且用户请求涉及交通、天气或路线推荐估算时，Agent 不得自行选择日期。
- Agent 应使用数字编号列出候选日期，让用户选择具体出发日。

## 验收

- Qwen Agent 系统提示应明确要求：周末出行必须询问清楚具体哪一天。
- Qwen Agent 系统提示应明确禁止：在交通、天气或路线推荐估算前自行选择周六或周日。
