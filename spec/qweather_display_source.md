# 天气来源展示规则

## 背景

天气工具可能返回和风天气官方预警聚合与每日天气预报。面向用户展示时需要明确数据来源，
避免用户误以为天气信息来自系统自行推断或未说明来源。

## 需求

- 当天气工具结果中的 `data_sources`、`official_alerts.source` 或 `daily_weather.source`
  表明数据来自和风天气时，回答中的天气参考必须明确写出来源。
- 官方预警和每日天气预报来源可分别说明，例如“官方预警来源：和风天气官方预警聚合”
  和“天气预报来源：和风天气每日天气预报”。
- 如果天气工具结果没有提供来源，只能说明“天气来源暂未提供”，不得自行补充来源。
- 来源展示只影响回答组织，不改变天气过滤、评分或推荐排序逻辑。
