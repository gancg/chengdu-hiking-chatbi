# Qwen 下一步动作编号选项

## 需求

当 Qwen 助手向用户征求下一步操作意见，并提供多个可选动作时，应使用连续数字
编号，让用户可以直接回复数字作出明确选择。

## 表达规则

- 多个下一步动作使用 `1. 2. 3.` 格式，通常提供 2–4 项。
- 明确提示用户可以直接回复数字。
- 不得使用图标、装饰符号或无编号项目符号代替数字编号。
- 每个选项必须是当前工具和数据能够完成的动作，不得承诺不存在的数据或能力。
- 普通需求访谈仍遵守每轮最多询问一个问题；编号选项只用于让用户选择下一步动作。

## 示例

```text
接下来可以继续：
1. 查询返程交通预估
2. 查询当天徒步时段的官方天气预警

直接回复 1 或 2 即可。
```

## 验收

- Qwen Agent 系统提示明确要求下一步动作使用数字编号。
- 系统提示明确允许用户直接回复数字。
- 系统提示禁止列出当前工具无法完成的选项。
