# sample_routes 权威导入

## 目标

显式执行 `init` 时，以 `data/sample_routes.json` 作为路线业务数据的唯一权威来源。普通应用启动只初始化数据库结构，不修改已有路线数据。

## 行为

- 完整读取并校验 `sample_routes.json` 后，才开始修改数据库。
- 在同一个事务中删除现有路线，再导入文件中的全部路线。
- 删除路线时，级联清理其交通、费用、停车点和行程反馈等关联数据。
- 清理或导入任一步骤失败时，事务必须回滚，不得留下半套数据。
- 普通 `import` 命令及服务内部的单批增量导入仍保留原有的更新/插入语义。
- 只有显式执行 `python -m hiking_chatbi init` 才进行权威导入；`serve`、Qwen 界面和组合应用启动均不得自动导入。
