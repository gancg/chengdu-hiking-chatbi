# 徒步 WebUI 清爽主题

## 目标

将默认 Qwen Gradio 页面调整为适合成都周边徒步问答的清爽、简洁界面，同时不改变对话、工具和推荐问题功能。

## 视觉规范

- 使用天空蓝、山林绿和大面积浅色留白。
- 助手头像使用山峰、登山杖、蓝天白云组成的徒步图标。
- 页面顶部展示简短的产品名称和用途说明。
- 对话区和右侧信息区使用轻边框、圆角和低强度阴影。
- 主按钮使用清晰的蓝绿色强调色，避免高饱和装饰和复杂背景。
- 小屏幕下左右栏允许自然换行，不产生横向滚动。

## 验收

- 业务配置明确指定徒步头像、页面标题和副标题。
- 头像文件存在于仓库资源目录。
- 主题 CSS 包含山野蓝配色、圆角卡片和响应式布局。
- 上传与音频模块继续保持关闭。
