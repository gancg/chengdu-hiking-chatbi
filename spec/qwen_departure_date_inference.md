# Qwen 出发日期年份推断

## 需求

用户提供出发日期但没有明确年份时，Qwen 徒步助手应将该日期理解为当前年度。

例如当前日期为 `2026-06-13`，用户说“6.15 号出发”，助手应按
`2026-06-15` 理解，并在调用工具时生成包含 `2026-06-15` 的 ISO 8601
出发时间。

## 设计

- 构建 Qwen Agent 时，将当前本地日期和当前年度注入系统提示。
- 系统提示明确要求：用户未明确年份时使用当前年度，不自行推断为下一年度。
- 用户明确给出年份时，以用户提供的年份为准。

## 验收

- 日期上下文包含当前日期、当前年度和无年份日期示例。
- Qwen Agent 的系统提示包含动态生成的日期上下文。
